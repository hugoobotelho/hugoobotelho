# 👨‍💻 Hugo Botelho

**`Desenvolvedor FullStack`**

Olá 👋🏻! Sejam bem-vindos. Me chamo Hugo, faço Ciência da Computação na Universidade Estadual do Sudoeste da Bahia (UESB) e, atualmente, estou no sexto semestre do curso.
